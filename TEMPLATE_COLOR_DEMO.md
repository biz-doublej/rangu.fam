## 사용법 요약

### 템플릿 레벨 색상 적용
```
{{템플릿명<bgcolor:#색상코드>
|키 = 값
}}

{{템플릿명<bgcolor:#배경색 color:#글자색>
|키 = 값
}}
```

### 지원하는 색상 형식
- **Hex 6자리**: `#FF0000`, `#00FF00`, `#0000FF`
- **Hex 3자리**: `#F00`, `#0F0`, `#00F`
- **프리셋 색상**: `blue-header`, `light-green`, `dark-blue` 등
- **투명**: `transparent`

### 지원하는 템플릿
- `{{인물정보상자<색상속성>`
- `{{그룹정보상자<색상속성>`
- `{{정보상자<색상속성>`

이 기능을 통해 템플릿 전체의 색상을 쉽게 변경하여 문서의 시각적 일관성을 높일 수 있습니다!
