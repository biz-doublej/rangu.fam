# 이랑위키 도움말 2025

모든 문서는 동일한 도구로 작성되고, 규칙만 지키면 누구나 차세대 UI를 즐길 수 있습니다. 아래는 최신 에디터 기능과 문법을 정리한 공식 가이드입니다.

---

## 1. 빠른 시작

1. **검색창에 제목 입력** → 결과가 없으면 “새 문서 작성” 버튼 노출  
2. **에디터에서 내용 작성** → `Ctrl+S` 로 저장, `Cmd/Ctrl+Enter` 로 미리보기 전환  
3. **문서 함수 패널**(에디터 하단)을 활용해 굵게/강조/아이콘 등을 한 번에 적용  
4. **저장 사유 입력** 후 “저장” → 자동으로 최신 문서가 생성됩니다.

> 💡 저장 중 충돌이 발생하면 편집 잠금 시스템이 알림을 띄우고, 10분 뒤 자동 해제됩니다.

---

## 2. 제목과 문단

| 위키 스타일 | 결과 |
| --- | --- |
| `= 대제목 =` | H1 |
| `== 중제목 ==` | H2 |
| `=== 소제목 ===` | H3 |
| `# Markdown` | 동일하게 지원 |

빈 줄 두 개 이상은 새 문단을 의미합니다. 문단 중간에서 `---` 또는 `***` 를 입력하면 구분선이 추가됩니다.

---

## 3. 텍스트 서식 & 문서 함수 패널

에디터 하단의 **문서 함수 패널**에서 버튼을 누르면 선택한 텍스트에 서식을 자동으로 적용할 수 있습니다.

> 📍 **패널 위치:** 문서가 열려 있는 동안 우측 사이드바의 [문서 함수 바로가기](#document-function-panel)를 눌러 패널로 즉시 이동할 수 있습니다.

| 버튼 | 적용되는 문법 | 설명 |
| --- | --- | --- |
| 굵게 | `'''텍스트'''` | 주요 용어 강조 |
| 기울임 | `''텍스트''` | 외래어/강조 |
| 밑줄 | `__텍스트__` | 링크와 혼동되지 않게 단락형 강조 |
| 취소선 | `~~텍스트~~` | 수정 기록 표시 |
| 인라인 코드 | `` `코드` `` | 짧은 명령어, 변수 |
| 색상 강조 | `{{{#ffeb3b 강조}}}` | HEX 색상 입력 후 “색상 적용” 버튼 |

색상 입력창 오른쪽의 프리셋 칩을 누르면 즉시 코드가 채워집니다. 각 버튼은 실행 후 좌측 하단에 “적용 완료” 메시지를 띄우며, 반복 클릭시 기존 서식을 덮어씁니다.

## 5. 링크·아이콘·이미지

| 문법 | 설명 |
| --- | --- |
| `[외부 링크](https://example.com)` | 새 창에서 열림 |
| `[[내부 문서]]` / `[[문서|표시 텍스트]]` | 이랑위키 문서 링크 |
| `!icon:{home}` | 기본 아이콘 |
| `[!icon:{insta} SNS](https://instagram.com)` | 아이콘+링크 조합 |
| `[!icon:{home,size:24,color:#ff0000}](/)` | 크기/색상 옵션 |
| `![설명](이미지URL)` | 인라인 이미지 |

업로드는 에디터 툴바의 이미지 버튼으로 진행합니다. 템플릿 내 이미지(인포박스, 카드그리드 등)는 자동으로 적절한 위치에 삽입됩니다.

## 6. 표와 컬러 테이블

### 기본 표
```text
| 제목1 | 제목2 |
|-------|-------|
| 내용1 | 내용2 |
```

### 색상 행
```text
||<bgcolor:#ff0000 color:#fff> 제목 ||
|| 일반 셀 ||
```

지원하는 스타일 키: `bgcolor`, `color`, `border`. HEX(#ff0000) 또는 사전 정의 값(예: `blue-header`)을 사용할 수 있습니다.

## 7. 대표 템플릿

### 7.1 인물 정보 (`{{인물정보}}`)
```text
{{인물정보
|이름 = 홍길동
|이미지 = /uploads/wiki/hong.jpg
|출생 = 1995-01-01
|국적 = 대한민국
}}
```

### 7.2 그룹 정보 (`{{그룹정보}}`)
```text
{{그룹정보
|그룹명 = Rangu.fam
|설립일 = 2024-03
|활동지역 = 서울
}}
```

### 7.3 카드 그리드
```text
[[카드그리드: items=[
  {"title":"프로젝트 A","description":"설명","image":"/uploads/wiki/a.jpg"}
]]]
```

각 템플릿의 색상을 바꾸려면 `{{템플릿명<bgcolor:#111827 color:#fff> ... }}` 처럼 `<>` 안에 속성을 추가합니다.

## 8. 문서 함수 UI 사용법

1. 적용할 텍스트를 드래그 → 패널 버튼 클릭  
2. 색상 강조는 HEX 입력 후 “색상 적용” 선택  
3. 패널 우측 하단 메시지에서 적용 결과 확인  
4. 동일한 스타일을 여러 번 누르면 가장 마지막 선택이 유지됩니다.

문서 함수 패널은 스크롤을 내리지 않아도 화면 우측 아래 고정된 플로팅 버튼(`↺ 서식`)으로 다시 열 수 있습니다.

---

## 9. 작업 흐름

### 9.1 새 문서 만들기
1. 제목 검색 → “새 문서 작성”  
2. 본문 입력 후 저장  
3. 문서 함수 패널로 서식 정리  
4. 저장 사유 입력 + 저장

### 9.2 기존 문서 편집
1. 우측 상단 “편집” 클릭  
2. 필요한 부분 수정 → `Ctrl+S`  
3. 편집 잠금 메시지가 뜨면 10분 대기 후 재시도

### 9.3 편집 잠금
- 다른 사용자가 작업 중이면 “편집 중” 알림 표시  
- 10분 동안 입력이 중지되면 자동 해제  
- 강제 해제는 관리자 메뉴에서만 가능

## 10. 운영 & 모범 사례

- **정확성**: 출처가 불분명한 내용은 각주([*1])와 함께 “추후 검증 필요” 주석을 남깁니다.  
- **중복 방지**: 동일한 문법 예시는 한 번만 설명하고, 세부 사례는 문서 함수 패널에서 해결합니다.  
- **아이콘 남용 금지**: 한 문단에 3개 이하로 제한합니다.  
- **표준 템플릿**: 인물/그룹/카드/문장 템플릿은 기본 형태를 유지하고 색상만 조정합니다.

<h2 id="분류-문법">11. 분류 문법</h2>

| 작성 예시 | 설명 |
| --- | --- |
| `분류:정치인` | 문서 본문 맨 마지막 줄에 적으면 자동으로 `정치인` 분류에 편입됩니다. |
| `분류:{정치\|한국 정치인}` | 중괄호를 사용해도 되며 `|` 로 여러 분류를 한 줄에서 구분할 수 있습니다. |
| `[[분류:정치인]]` | 링크 문법을 쓰면 클릭 가능한 배지가 만들어지고 `/wiki/category/정치인` 페이지로 이동합니다. |

- 분류 한 줄당 하나 이상의 분류를 선언할 수 있으며, 여러 줄을 작성하면 각각 적용됩니다.
- 분류 배지를 클릭하면 상단 광고/카운터를 포함한 고정형 분류 페이지가 열리며 하위 분류가 자동 정렬됩니다.
- 상위/하위 구조가 필요한 경우 `[[분류:최상위]]`, `[[분류:하위]]`처럼 복수의 분류를 함께 기재하세요.

## 12. FAQ

| 질문 | 답변 |
| --- | --- |
| 링크가 안 먹어요 | `[텍스트](https://example.com)` 형식을 확인하세요. `<a>` 태그는 지원하지 않습니다. |
| 다른 사람이 편집 중이면? | 상단에 “XX님이 편집 중” 배지가 나타납니다. 10분 후 자동 해제되며, 강제 저장은 차단됩니다. |
| 이미지 용량 제한 | 10MB 이하 JPG/PNG/WebP. GIF는 5MB. |
| 템플릿 색상이 이상해요 | `bgcolor`에 유효한 HEX 또는 사전 정의 이름을 사용했는지 확인하세요. |
| 자동 저장은 언제 되나요? | 2분마다 초안이 로컬에 저장되며, 새 창에서도 복구 알림이 나타납니다. |
